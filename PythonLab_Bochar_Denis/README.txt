	
	Программа выполняется из коммандной строки 

путем запуска файла index.js d окружениb node.js//

path>node index.js btcAddress.txt

Текстовый файл btcAddress.txt является аргументом для index.js,

в который предварительно нужно записать биткоин-адрес.

Программа реализована таким способом, потому что биткоин-адрес 

представляет собой последовательность из 25-34 символов Base58,

а так как в коммандной строке механизм copy/past на OS Windows

не предусмотрен(точнее он есть, но очень неудобен),

всегда сувществует большая вероятность ошибиться при вводе данных 

вручную.
